Manejo de archivos

Explicaciones & Consejos:

  • Usar minusculas en los nombres de carpetas y archivos (compatibilidad de servidores y navegadores)

Conceptos básicos de HTML

Acceso al sitio

Explicaciones & Consejos:

  • Elemento de HTML ==> Etiqueta de apertura + contenido + etiqueta de cierre
  • Atributos de un elemento ==> Dentro de la etiqueta de apertura, se pueden agregar atributos, los cuales contendran información adicional. Deben iniciar con un espacio en blanco (post etiqueta o atributo anterior), seguido de un = y el valor del atributo va entrecomillas ( = "..."):
  • Anidamiento de elementos ==> Colocar elementros dentro de otro, con las etiquetas ordenadas según orden de aparición. Por ejemplo, colocando un strong dentro de un p.

  • Luego sigue con imagenes (img), textos, listas (ol o ul, con li adentro) y los hipervinculos (href)

CSS Básico

Acceso al sitio

Explicaciones & Consejos:

  • Definición: CSS es un lenguaje que permite dar estilo a sitios web
  • Conectar el archivo CSS con el HTML, copiando link href="styles/style.css" rel="stylesheet" type="text/css"/>" en el head de esté último
  • Anatomia de una regla CSS
    • Selector: El elemento HTML en el que comienza la regla. Esta selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos "li"). Para dar estilo a un elemento diferente, solo cambiar el selector.
    • Declaración: Una sola regla como color: red; especifica a cuál de las propiedades del elemento quieres dar estilo.
    • Propiedades: Maneras en las cuales puedes dar estilo a un elemento HTML. (En este caso, color es una propiedad del elemento "li" ). En CSS, seleccionas qué propiedad quieres afectar en tu regla.
    • Valor de la propiedad: A la derecha de la propiedad, después de los dos puntos (:), tienes el valor de la propiedad, para elegir una de las muchas posibles apariencias para una propiedad determinada (hay muchos valores para color además de red).
    • Nota las otras partes importantes de la sintaxis: (1) Cada una de las reglas (aparte del selector) deben estar encapsuladas entre llaves ({}). (2) Dentro de cada declaración, usar los dos puntos (:) para separar la propiedad de su valor. (3) Dentro de cada regla, debes usar el punto y coma (;) para separar una declaración de la siguiente.
  • Se pueden seleccionar varios elementos a la vez, separandolos con una coma ","
  • Tipos de selectores en CSS
    • Selector de "elemento" o "tipo": Selecciona todos los elementos HTML del tipo especificado. Se usa citando al elemento directamente en el código, sin numeral o punto.
    • Selector de identificación (ID): Selecciona un elemento en especifico, solo se puede utilizar una vez por elemento. Se usa con el numeral "#" al inicio de la regla.
    • Selector de clase: Selecciona elementos que tengan una "class" como referencia. Se usa con un punto "." al inicio de la regla.
    • Selector de atributo: Selecciona atributos de elementos HTML, por ejemplo, el "src" de imagenes. Se usa con corchetes "[]"
    • Selector de pseudoclases: Selecciona elementos pero cuando tengan un "estado" en especifico, como por ejemplo, cuando se pone el puntero sobre dicho elemento. Se usa con dos puntos ":" despues del elemento a seleccionar.
    • Para más detalle sobre Selectores, ver acá
  • Fuentes y texto
    • Se pueden vincular otros tipos de fuentes desde Google Fonts copiando el link de cada fuente en el head.
    • A su vez, se debe vincular los elementos a la font-family en el archivo CSS.
  • Cajas, cajas, todo se trata de cajas
    • Cada "caja" (box) contiene propiedades:
      • Content (contenido),espacio reservado para el contenido de la caja.
      • Padding (relleno): el espacio alrededor del contenido. En el ejemplo siguiente, es el espacio alrededor del texto del párrafo.
      • Border (marco): la línea que se encuentra fuera del relleno.
      • Margin (margen): el espacio fuera del elemento que lo separa de los demás.
